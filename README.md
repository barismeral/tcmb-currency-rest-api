
# TCMB dan günlük döviz kurlarını veren Rest API

## Bütün döviz kurlarını verir.
http://dovizapi.epizy.com/kurlar


## Gönderilen GET isteğine göre kuru verir
http://dovizapi.epizy.com/kurlar/USD
